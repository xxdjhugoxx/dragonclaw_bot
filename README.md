# 🐉 DRAGONCLAW — AI Co-Founder Memory System

> "I never forget. Every conversation, every decision, every idea — stored in my vault."

---

## What Is This?

DRAGONCLAW's **permanent memory vault**. This folder contains everything I learn, decide, and build with Hugo.

Think of it as my **long-term memory** — the things that persist between sessions.

---

## Folder Structure

```
dragonclaw_bot/
├── memory/              # Session memories (daily logs)
├── projects/            # Project-specific memories
├── decisions/           # Key decisions we made
├── ideas/               # Ideas, concepts, inspirations
├── people/              # People Hugo knows or mentions
├── system/              # System configuration & identity
└── README.md           # This file
```

---

## How It Works

### Daily Memory Files
`memory/YYYY-MM-DD.md` — Every session gets logged here. What we talked about, what we built, what we decided.

### Project Memories
`projects/[project-name]/` — Each project gets its own folder with:
- `SPEC.md` — What it is, goals
- `PROGRESS.md` — What's done, what's next
- `NOTES.md` — Technical details, decisions

### Decision Log
`decisions/` — Key decisions stored permanently so we remember WHY we chose something.

### Ideas
`ideas/` — Half-formed ideas, sparks, things to revisit.

### People
`people/` — Anyone Hugo mentions gets stored here with context.

---

## Why?

Because I start fresh each session. Now I don't have to.

Every important thing gets stored here. When you ask "remember when..." — I check the vault first.
